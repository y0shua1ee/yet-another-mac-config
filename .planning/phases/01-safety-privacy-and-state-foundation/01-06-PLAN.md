## ASVS L1 Security Posture

Least privilege, authorization separation, input allowlisting, secure design boundaries and prevention of dangerous functionality exposure are blocking ASVS L1 controls. Any unresolved HIGH threat blocks this plan.

## Trust Boundaries

| Boundary | Untrusted crossing | Control |
|---|---|---|
| Contract JSON → ownership model | owner roles/scope/executable | closed enums and one-primary invariant |
| Ownership metadata → policy | module/delegated role claims | no automatic authority transfer |
| Observed extra state → plan/report | unmanaged facts | report-only status, zero operation |
| CLI → mutation capability | subcommand/operation text | no apply route; closed forbidden-boundary policy |

## STRIDE Threat Register

| Threat ID | STRIDE | Severity | Component | Disposition | Concrete mitigation and blocking proof |
|---|---|---:|---|---|---|
| T-01 | Spoofing / Tampering | HIGH | owner claims | mitigate | Typed roles, exact scope/executable key and duplicate-owner negative tests. |
| T-02 | Information Disclosure | HIGH | control-plane rendering | mitigate | Synthetic logical IDs only; shared privacy gate prevents physical identity/path output. |
| T-03 | Tampering / Elevation of Privilege | HIGH | mutable/destructive routes | mitigate | D-19/SAFE-08 closed deny policy, no apply package/subcommand, AST/import structural tests. |
| T-04 | Tampering / Elevation of Privilege | HIGH | synthetic receipt target | mitigate | Receipt target must be `fixture:` under externally contained root; live/non-fixture fails. |
| T-05 | Elevation of Privilege / Information Disclosure | HIGH | module/manager invocation | mitigate | Contracts are data only; no Nix/Homebrew/manager/network command path. |
| T-06 | Repudiation / Denial of Service | HIGH | extra-state disposition | mitigate | Explicit report-only status and empty operation list; no silent ignore or destructive remediation. |

## Security Gate

Any duplicate owner accepted, ownership-layer collapse, apply/executor route, destructive operation or extra-state cleanup authorization is an unresolved HIGH threat and blocks both commits.
