## ASVS L1 Security Posture

All external inputs are untrusted; schema/input validation, deny-by-default command routing, safe output handling, filesystem containment, deterministic evidence and least privilege are blocking ASVS L1 controls. Any unresolved HIGH threat below blocks this plan.

## Trust Boundaries

| Boundary | Untrusted crossing | Required control in this plan |
|---|---|---|
| CLI → workflow | subcommand, blueprint and root arguments | closed command set, canonical containment, no shell dispatch |
| Tracked blueprint → typed artifacts | synthetic JSON input | fixed schema shape for the happy path, logical references only |
| Workflow → external store | canonical artifact bytes | recomputed SHA-256 key, atomic external write |
| Fake adapter → evidence | fixture-owned mutation/result | synthetic mode marker and fresh observation |
| Sentinel → report | before/after tokens and manifest identity | exact binding and scoped claim only |

## STRIDE Threat Register

| Threat ID | STRIDE | Severity | Component | Disposition | Concrete mitigation and blocking proof |
|---|---|---:|---|---|---|
| T-01 | Spoofing / Tampering | HIGH | artifact kind and lineage | mitigate | D-01/D-04 closed kinds, recomputed SHA-256, exact edge assertions; walking-skeleton test rejects store-key mismatch and missing edge. |
| T-02 | Information Disclosure | HIGH | CLI/store output | mitigate | Inputs are fully synthetic and renderer exposes only fixed logical IDs/digests; E2E rejects canonical root leakage. Plan 01-03 adds the general pre-output gate before non-synthetic adapter contracts exist. |
| T-03 | Tampering / Elevation of Privilege | HIGH | operation routing | mitigate | No apply package/subcommand, no arbitrary shell or live adapter, fake receipt is explicitly synthetic and fixture-scoped; structural assertions block mutation routes. |
| T-04 | Tampering / Elevation of Privilege | HIGH | fixture filesystem and environment | mitigate | Fresh external roots, canonical repo exclusion, allowlisted environment and isolated HOME/XDG/cache roots; escape test must pass. |
| T-05 | Elevation of Privilege / Information Disclosure | HIGH | toolchain/network escalation | mitigate | Local Go only, proxy/sum DB disabled, no network/live flags and no manager bootstrap; missing toolchain is manual-required. |
| T-06 | Repudiation / Denial of Service | HIGH | sentinel evidence and claim ceiling | mitigate | Complete before/after synthetic evidence binds exact manifest/window; missing evidence is non-zero, the sole success state is `synthetic-sentinel-passed`, and E2E negatives reject every real-surface/whole-Mac/current-host claim. |

## Security Gate

- No HIGH threat may remain accepted, transferred, or unresolved.
- RED must fail for the intended absent behavior; GREEN must pass inside the external fixture.
- Any privacy leak, in-worktree write, network attempt, live command route, incomplete sentinel evidence or stronger claim blocks commit and plan completion.
