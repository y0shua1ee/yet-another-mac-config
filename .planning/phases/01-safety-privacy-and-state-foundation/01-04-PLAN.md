**As a** Mac 配置仓库操作者，**I want to** 校验严格区分且有 lineage 的恢复 artifacts，并在隔离 fixture 中运行端到端安全测试，**so that** 我能证明本次运行覆盖的真实 Mac 表面没有发生未经授权的变化。

<objective>
把 walking skeleton 的 external temp usage 固化为可复用 fixture lifecycle，并加入不可升级的三-tier policy、exact network authorization contract 与 live-check deny/manual behavior。

Purpose: 完整落实 SAFE-04～SAFE-06 与 D-09～D-12，使测试缺能力、cache miss 或失败永远不会转成安装、下载、trust、network 或 live probe。

</objective>

## ASVS L1 Security Posture

Filesystem containment, least privilege, environment isolation, network deny-by-default, secure temporary storage and explicit authorization are blocking ASVS L1 controls. Any unresolved HIGH threat blocks this plan.

## Trust Boundaries

| Boundary | Untrusted crossing | Control |
|---|---|---|
| CLI → fixture base | operator path/tier/retention flags | canonical containment and closed options |
| Parent process → child env | ambient variables/PATH | blank allowlist plus external roots |
| Retention request → deletion | logical ID/marker/path | UID+nonce+TTL+schema+non-symlink+one-child checks |
| Network/live request → capability | test/probe ID and manifest | exact tracked contract; no executor in Phase 1 |

## STRIDE Threat Register

| Threat ID | STRIDE | Severity | Component | Disposition | Concrete mitigation and blocking proof |
|---|---|---:|---|---|---|
| T-01 | Spoofing / Tampering | HIGH | fixture marker/network manifest | mitigate | Closed schema, digest/ID checks and ownership nonce; malformed/substituted metadata is non-zero. |
| T-02 | Information Disclosure | HIGH | environment/retained fixture | mitigate | Blank allowlist, logical output and no live raw/credential/proxy retention. |
| T-03 | Tampering / Elevation of Privilege | HIGH | fixture teardown and tier routing | mitigate | Post-verdict one-child marker/UID/nonce/containment teardown is the sole cleanup and combines monotonically; closed tier enum cannot dispatch restore, convergence, apply or live mutation. |
| T-04 | Tampering / Elevation of Privilege | HIGH | fixture filesystem | mitigate | D-10 canonical external containment, full root isolation and symlink/traversal negative tests. |
| T-05 | Elevation of Privilege / Information Disclosure | HIGH | network/live capability | mitigate | D-09/D-12 exact ID, deny default, no HTTP client, no broad flag, manual-required when unenforceable. |
| T-06 | Repudiation / Denial of Service | HIGH | lifecycle outcomes | mitigate | Marker timestamps/identity, deterministic status, timeout/byte bounds and no automatic retry/escalation. |

## Security Gate

Any write outside the fixture, inherited secret/proxy variable, ambiguous cleanup, actual egress, live probe execution or tier escalation is an unresolved HIGH threat and blocks both commits.
