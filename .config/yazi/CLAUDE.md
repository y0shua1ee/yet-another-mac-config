# Guidance for agents

## Introduction and Structure
- This folder contains the configuration files for yazi and its plugins. The official address for yazi is https://yazi-rs.github.io/.
- All configuration files are configured here as .toml files. The plugins folder contains yazi plugins. The .toml configuration files and init.lua files are configured according to the official yazi documentation and the documentation for each plugin.

## Workflow
- After configuration, write the steps required by the plugin into the `install_yazi_plugins.sh` script in the project root.
